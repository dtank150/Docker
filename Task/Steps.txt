Task 1: Create 3 network in docker with name of dev, uat and prod.
Each network have 1 container and all container can communicate with each other. 

Create 3 Network using:-
docker network create command:-
docker network create dev
docker network create uat
docker network create prod

For Container:-
docker run -d --name dev_container --network dev hello-world
docker run -d --name uat_container --network uat hello-world
docker run -d --name prod_container --network prod hello-world

For Communictae:-
docker exec dev_container ping uat_container
docker exec uat_container curl prod_container:8080
